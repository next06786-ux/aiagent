# 知识图谱可视化 - 升级对比

## 🎯 升级前后对比

### 视觉效果对比

| 方面 | 升级前 | 升级后 |
|------|--------|--------|
| **节点效果** | 简单渐变圆形 | 脉冲 + 光晕 + 多层阴影 |
| **连接线** | 静态渐变线 | 流动光点 + 动态渐变 |
| **背景** | 纯色背景 | 多色渐变 + 粒子系统 |
| **选中状态** | 双层光环 | 多层光环 + 平滑动画 |
| **详情面板** | 基础毛玻璃 | 增强毛玻璃 + 流畅动画 |
| **整体风格** | 简洁风格 | 鸿蒙6官网风格 |

### 动画效果对比

#### 升级前
```
节点：静态显示
连接：静态线条
背景：纯色
交互：基础反馈
```

#### 升级后
```
节点：脉冲 + 光晕 + 阴影
连接：流动光点 + 渐变
背景：渐变 + 粒子
交互：多层反馈 + 平滑过渡
```

## 🎨 设计系统应用

### 颜色系统
- **主色**：#0A59F7（鸿蒙蓝）
- **辅色**：#6B48FF（星空紫）
- **强调色**：#00C853（生机绿）
- **背景**：#F5F7FA（主背景）

### 渐变系统
- **Aurora**：极光渐变 #0A59F7 → #6B48FF
- **Nebula**：星云渐变 #8E2DE2 → #4A00E0
- **Glass**：玻璃渐变 rgba(255,255,255,0.8) → rgba(255,255,255,0.4)

### 阴影系统
- **Soft**：柔和阴影（12px, 0.04 opacity）
- **Medium**：中等阴影（20px, 0.06 opacity）
- **Elevated**：提升阴影（40px, 0.12 opacity）

## 🔄 动画时序

### 脉冲动画
```
周期：连续循环
幅度：±15% 缩放
速度：0.02 弧度/帧
效果：节点呼吸感
```

### 流动动画
```
周期：连续循环
速度：0.03 弧度/帧
光点大小：8px
光点强度：0.8 → 0
效果：知识流动感
```

### 粒子动画
```
数量：50个
生命周期：50-150帧
运动：随机速度
效果：环境氛围
```

## 📊 性能指标

### 渲染性能
- **帧率**：60 FPS（目标）
- **Canvas 大小**：360×600px
- **节点数量**：建议 5-20 个
- **连接数量**：建议 < 30 条

### 内存占用
- **粒子系统**：~50KB
- **节点数据**：~1KB/节点
- **Canvas 缓冲**：~864KB

### 优化策略
1. 裁剪区域限制绘制范围
2. 收敛检测自动停止模拟
3. 速度限制防止过度运动
4. 帧率控制在 60fps

## 🔧 自定义选项

### 可配置参数
```typescript
// 物理参数
REPULSION: 3000        // 斥力强度
ATTRACTION: 0.05       // 引力强度
DAMPING: 0.85         // 阻尼系数

// 动画参数
GLOW_ANIMATION_SPEED: 0.05  // 光晕速度
PARTICLE_COUNT: 50          // 粒子数量

// 视觉参数
NODE_RADIUS: 24             // 节点半径
GLOW_RADIUS: 12             // 光晕半径
```

### 主题定制
```typescript
// 修改 Theme.ets 中的颜色
Theme.Color.Primary = '#0A59F7'      // 主色
Theme.Color.Secondary = '#6B48FF'    // 辅色
Theme.Gradient.Aurora = ['#0A59F7', '#6B48FF']  // 渐变
```
