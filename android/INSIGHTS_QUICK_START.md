# 智慧洞察功能 - 快速开始

## 功能概述
智慧洞察是一个专业的 AI 分析系统，包括三个专业 Agent（人际关系、教育升学、职业规划）和跨领域综合分析功能。基于 RAG 记忆系统和 Neo4j 知识图谱，提供深度智能分析和战略建议。

## 当前实现状态

### ✅ 已完整实现
- **单 Agent 分析** - 三个专业 Agent 提供深度洞察报告
- **跨领域综合分析** - 多 Agent 协作，发现跨领域关联和协同效应
- **视图模式切换** - 在单 Agent 分析和跨领域分析之间自由切换

### ⏳ 计划中
- 报告分享和收藏功能
- 报告历史记录
- 数据可视化图表
- 离线缓存

## 使用流程

### 1. 进入智慧洞察
- 打开应用 → 主页 → 点击"智慧洞察"卡片
- 进入智慧洞察主界面

### 2. 选择分析模式
智慧洞察提供两种分析模式：

#### 模式 A：单 Agent 分析
专注于单一领域的深度分析

**步骤**：
1. 点击"单Agent分析"按钮
2. 选择一个专业 Agent：
   - 👥 **人际关系 Agent** - 分析人际关系网络、社交模式、关系质量
   - 🎓 **教育升学 Agent** - 分析升学路径、学校选择、专业匹配
   - 💼 **职业规划 Agent** - 分析职业发展、技能匹配、岗位选择
3. 点击"查看报告"按钮
4. 等待 Agent 分析（通过 RAG 和 Neo4j 混合检索）
5. 查看完整报告

**报告内容**：
- **AI 智能摘要** - Agent 对整体情况的综合分析
- **关键发现** - 重要的发现和洞察（带重要性标识）
- **专业建议** - 具体可执行的建议（带优先级和时间线）
- **决策逻辑** - 推理路径和影响因素分析
- **数据来源** - RAG 记忆系统和 Neo4j 知识图谱的节点数量

#### 模式 B：跨领域综合分析
多 Agent 协作，发现跨领域关联

**步骤**：
1. 点击"跨领域分析"按钮
2. 输入分析查询（或选择快捷查询）
3. 点击"开始分析"按钮
4. 等待多 Agent 协作分析
5. 查看综合分析结果

**分析结果包含 4 个标签页**：
- **综合摘要** - 整体分析和整合洞察
- **跨领域模式** - 模式识别、协同效应、潜在冲突
- **战略建议** - 优先级排序的综合建议
- **行动计划** - 短期/中期/长期分阶段计划

### 3. 快捷查询（跨领域分析）
系统提供常用查询模板：
- "综合分析我的人际关系、教育背景和职业发展"
- "我的人际关系如何影响职业发展？"
- "教育背景和人脉资源如何协同提升职业竞争力？"
- "如何平衡学业、人际关系和职业规划？"

### 4. 返回和切换
- 在 Agent 报告界面，点击"返回Agent选择"可重新选择 Agent
- 随时切换视图模式（单Agent / 跨领域）

## 跨领域分析说明

### 跨领域模式
识别不同领域之间的关联模式，例如：
- 人际关系如何影响职业发展
- 教育背景如何影响人脉资源
- 职业发展如何反馈到学习需求

### 协同效应
发现多个领域的正向互动，例如：
- 人脉资源 × 职业发展 = 更多机会
- 教育背景 × 人际关系 = 高质量社交圈
- 职业技能 × 学习能力 = 快速成长

### 潜在冲突
识别领域间的冲突和矛盾，例如：
- 工作时间 vs 社交时间
- 短期收益 vs 长期发展
- 专业深度 vs 技能广度

### 战略建议
基于跨领域分析的综合建议，带优先级和时间线。

### 行动计划
分阶段的具体行动计划：
- **短期（1-3个月）** - 立即可执行的行动
- **中期（3-6个月）** - 需要准备的行动
- **长期（6-12个月）** - 战略性的行动

## 重要性和优先级说明

### 重要性（关键发现）
- **🔴 High** - 高度重要，需要立即关注
- **🟠 Medium** - 中等重要，需要注意
- **🟢 Low** - 一般重要，可以了解

### 优先级（专业建议）
- **🔴 High** - 高优先级，建议优先执行
- **🟠 Medium** - 中优先级，可以安排执行
- **🟢 Low** - 低优先级，有时间再执行

## 常见问题

### Q: Agent 分析需要多长时间？
A: 单 Agent 分析通常需要 10-30 秒，跨领域分析可能需要更长时间（30-60秒），因为涉及多 Agent 协作。

### Q: 为什么 Agent 报告内容不同？
A: 每个 Agent 专注于不同领域，使用不同的分析模型和数据源，因此报告内容会有差异。

### Q: 置信度分数是什么意思？
A: 置信度表示 Agent 对分析结果的信心程度，范围 0-1（0-100%）。置信度越高，分析结果越可靠。

### Q: 如何提高分析质量？
A: 多使用系统功能，积累更多行为数据。系统会自动将数据存入 RAG 记忆系统和 Neo4j 知识图谱，从而提高分析质量。

### Q: 跨领域分析和单 Agent 分析有什么区别？
A: 单 Agent 分析专注于单一领域的深度分析；跨领域分析通过多 Agent 协作，发现不同领域之间的关联、协同效应和潜在冲突，提供更全面的战略建议。

### Q: 可以保存报告吗？
A: 当前版本暂不支持报告保存功能，后续版本会添加报告收藏和历史记录功能。

### Q: 数据来源中的 RAG 和 Neo4j 是什么？
A: 
- **RAG（检索增强生成）** - 记忆系统，存储用户的对话、决策等行为数据
- **Neo4j** - 知识图谱数据库，存储实体和关系的网络结构
- Agent 通过混合检索这两个数据源，生成更准确的分析报告
