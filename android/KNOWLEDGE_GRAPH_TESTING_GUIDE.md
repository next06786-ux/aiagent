## 已知限制

1. **文字不随 3D 场景旋转**
   - 文字始终面向用户（Billboard 效果）
   - 这是 Compose Overlay 方案的特性，不是 bug

2. **文字可能重叠**
   - 当节点距离很近时，文字可能重叠
   - 未来可以添加碰撞检测和文字避让

3. **性能**
   - 文字层每秒更新 60 次
   - 在节点数量很多时（>100）可能影响性能
   - 可以通过降低更新频率优化

## 如果还有问题

**检查**：
1. 查看日志是否有错误
2. 确认 `viewWidth` 和 `viewHeight` 不为 0
3. 确认 `nodes` 列表不为空

**解决**：
```bash
adb logcat | grep "NodeLabelsOverlay"
```

**检查**：
1. 查看缩放日志
2. 确认 `getCameraDistance()` 返回正确值

**解决**：
```bash
adb logcat | grep "缩放"
```

