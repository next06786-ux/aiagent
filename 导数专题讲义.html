导数专题系统讲义

概念、方法与技巧全解析（含详细答案）

01

概念与求导法则

切入点与着手点

  • 1. 看到 $f'(a)$ 出现在函数表达式中，要意识到它是常数，设为 $k$ 后代入求解
  • 2. 切线问题：先求导得斜率，再用点斜式写方程
  • 3. 注意区分"在点处的切线"与"过点的切线"

【例题】

设函数 $f(x) = x^2 - xf'(1)$，则曲线 $y = f(x)$ 在点 $(2, f(2))$ 处的切线方程为（    ）

A. $3x - y - 4 = 0$    B. $3x - y + 4 = 0$    C. $3x + y + 4 = 0$    D. $3x + y - 4 = 0$

【答案】A

解析：关键注意 $f'(1)$ 是一个常数。

1. 求导得 $f'(x) = 2x - f'(1)$

2. 令 $x = 1$：$f'(1) = 2 - f'(1)$，解得 $f'(1) = 1$

3. 所以 $f(x) = x^2 - x$，$f(2) = 2$，$f'(x) = 2x - 1$

4. 斜率 $k = f'(2) = 3$，切线方程：$y - 2 = 3(x - 2)$

5. 整理得：$3x - y - 4 = 0$

02

单调性判定

切入点与着手点

  • 1. 求导 → 令 $f'(x) = 0$ 找零点 → 列表或画图判断正负
  • 2. 若 $f'(x)$ 是分式，分母恒正时只看分子；分母可变号时需讨论
  • 3. 二次函数型导数：看判别式 $\Delta$ 决定是否有零点
  • 4. 定义域是前提，零点必须在定义域内才有效

【例题】

已知函数 $f(x) = \ln x - ax + x^2$，讨论 $f(x)$ 的单调性。

【答案】

定义域为 $(0, +\infty)$，$f'(x) = \dfrac{1}{x} - a + 2x = \dfrac{2x^2 - ax + 1}{x}$

令 $g(x) = 2x^2 - ax + 1$，判别式 $\Delta = a^2 - 8$：

  • 当 $-2\sqrt{2} \leq a \leq 2\sqrt{2}$ 时，$\Delta \leq 0$，$f'(x) \geq 0$，函数在 $(0, +\infty)$ 上单调递增；
  • 当 $a < -2\sqrt{2}$ 或 $a > 2\sqrt{2}$ 时，$\Delta > 0$，有两根 $x_{1,2} = \dfrac{a \pm \sqrt{a^2-8}}{4}$。根据两根正负及大小关系进一步讨论增减区间（需保证根在定义域内）。
03

含参讨论单调性

切入点与着手点

  • 1. 在指定区间单调：$f'(x) \geq 0$（或 $\leq 0$）在该区间恒成立
  • 2. 转化为求最值问题：$f'(x)_{\min} \geq 0$ 或 $f'(x)_{\max} \leq 0$
  • 3. 注意对称轴与区间的位置关系
  • 4. 交点问题：转化为方程根的个数问题

【例题】

已知 $f(x) = \dfrac{1}{3}x^3 + ax$，$g(x) = -x^2 - a$ （$a \in \mathbb{R}$）。

  • (1) 若 $F(x) = f(x) - g(x)$ 在 $[1, +\infty)$ 上单调递增，求 $a$ 的最小值；
  • (2) 若 $G(x) = f(x) + g(x)$ 的图像与 $y = ax$ 有且只有一个交点，求 $a$ 的取值范围。

【答案】(1) $-3$；(2) $\left(-\infty, -\dfrac{3}{4}\right)$

(1) $F(x) = \dfrac{1}{3}x^3 + x^2 + ax + a$，$F'(x) = x^2 + 2x + a \geq 0$ 在 $[1, +\infty)$ 恒成立。

对称轴 $x = -1$，故 $F'(x)$ 在 $[1, +\infty)$ 递增，只需 $F'(1) = 3 + a \geq 0$，得 $a \geq -3$。

(2) 方程 $\dfrac{1}{3}x^3 - x^2 = ax$ 即 $x\left(\dfrac{1}{3}x^2 - x - a\right) = 0$。

已有根 $x = 0$，需方程 $\dfrac{1}{3}x^2 - x - a = 0$ 无实根或根为0。

若无实根：$\Delta < 0$ 即 $1 + \dfrac{4}{3}a < 0$，得 $a < -\dfrac{3}{4}$。

若根为0：代入得 $a = 0$，此时另一根为3，有两个交点，舍去。

04

求参问题：全分离参数

切入点与着手点

  • 1. 全分离：将不等式变形为 $a < h(x)$ 或 $a > h(x)$ 的形式
  • 2. 恒成立问题：$a < h(x)$ 恒成立 $\Leftrightarrow$ $a < h(x)_{\min}$
  • 3. 分离时注意讨论分母的正负（可能需要分区间讨论）
  • 4. 特殊点（如使分母为0的点）要单独验证

【例题】

已知 $f(x) = e^x - ax + 2a > 0$ 恒成立，求 $a$ 的取值范围。

【答案】$[0, e^3)$

解析：变形为 $e^x > a(x - 2)$。

  • 当 $x = 2$ 时，$e^2 > 0$ 恒成立；
  • 当 $x > 2$ 时，$a < \dfrac{e^x}{x-2}$。令 $g(x) = \dfrac{e^x}{x-2}$，求导得极小值点 $x = 3$，$g(3) = e^3$，故 $a < e^3$；
  • 当 $x < 2$ 时，$a > \dfrac{e^x}{x-2}$。因 $h(x) = \dfrac{e^x}{x-2}$ 在 $(-\infty, 2)$ 上值域为 $(-\infty, 0)$，故 $a \geq 0$。

综上：$0 \leq a < e^3$。

05

求参问题：半分离参数

切入点与着手点

  • 1. 半分离：当全分离后函数过于复杂时，只分离出参数系数
  • 2. 形如 $m \leq \dfrac{f(x)}{g(x)}$，其中 $g(x)$ 可能变号
  • 3. 先处理特殊点（使 $g(x)=0$ 的点），再分区间讨论
  • 4. 求导时注意商的求导法则

【例题】

已知 $f(x) = e^x \cdot \cos x$，$g(x) = \sin x + 1$，当 $x \in \left[-\dfrac{\pi}{2}, 0\right]$ 时，若 $f(x) \geq mg(x)$ 恒成立，求 $m$ 的取值范围。

【答案】$(-\infty, 1]$

解析：当 $x = -\dfrac{\pi}{2}$ 时，左边$=0$，右边$=0$，成立。

当 $x \in \left(-\dfrac{\pi}{2}, 0\right]$ 时，$\sin x + 1 > 0$，分离得 $m \leq \dfrac{e^x \cos x}{1 + \sin x}$。

令 $h(x) = \dfrac{e^x \cos x}{1 + \sin x}$，求导化简得 $h'(x) = \dfrac{e^x \cdot (-(\cos x - 1)^2)}{2(1+\sin x)^2} \leq 0$。

故 $h(x)$ 在区间上单调递减，最小值为 $h(0) = 1$，所以 $m \leq 1$。

06

切线方程梳理

切入点与着手点

  • 1. "在点$(a,f(a))$处的切线"：切点已知，直接求 $k=f'(a)$，用点斜式
  • 2. "过点$(a,b)$的切线"：设切点 $(t, f(t))$，列方程组求 $t$
  • 3. 先验证点是否在曲线上，决定用哪种方法
  • 4. 公切线问题：设切点，利用斜率相等和点在直线上列方程

【例题】

函数 $f(x) = x \ln x + 2x^3$，则曲线 $y = f(x)$ 在点 $(1, 2)$ 处的切线方程是__________。

【答案】$7x - y - 5 = 0$（或 $y = 7x - 5$）

解析：

1. 验证：当 $x = 1$ 时，$f(1) = 1 \cdot 0 + 2 = 2$，点在曲线上。

2. 求导：$f'(x) = \ln x + 1 + 6x^2$

3. 斜率：$k = f'(1) = 0 + 1 + 6 = 7$

4. 点斜式：$y - 2 = 7(x - 1)$，整理得 $7x - y - 5 = 0$。

07

端点效应

切入点与着手点

  • 1. 端点效应：若 $f(a) = 0$ 且要证 $f(x) \geq 0$（$x > a$），则必有 $f'(a) \geq 0$
  • 2. 这是必要条件，可快速缩小参数范围
  • 3. 得到必要条件后，还需验证充分性
  • 4. 适用于不等式恒成立且端点取等的情况

【例题】

不等式 $x \ln x \geq a(x - 1)$ 对任意 $x > 1$ 恒成立，求 $a$ 的范围。

【答案】$(-\infty, 1]$

解析：设 $f(x) = x \ln x - a(x - 1)$，注意到 $f(1) = 0$。

求导得 $f'(x) = \ln x + 1 - a$。

由端点效应，若 $f(x) \geq 0$ 在 $x > 1$ 恒成立，则必有 $f'(1) \geq 0$（否则在1右侧附近函数值会小于0）。

$f'(1) = 1 - a \geq 0$，得必要条件 $a \leq 1$。

充分性验证：当 $a \leq 1$ 时，$f'(x) = \ln x + 1 - a \geq \ln x > 0$（$x > 1$），故 $f(x)$ 单调递增，$f(x) > f(1) = 0$，成立。

08

指数找朋友，对数单身狗

切入点与着手点

  • 1. 核心思想：利用切线放缩，将 $e^x$ 和 $\ln x$ 分别放缩
  • 2. 指数放缩：$e^x \geq x + 1$（切线在 $x=0$ 处，当且仅当 $x=0$ 取等）
  • 3. 对数放缩：$\ln x \leq x - 1$（切线在 $x=1$ 处，当且仅当 $x=1$ 取等）
  • 4. 若取等条件不同时满足，则不等式严格成立
  • 5. 适用于证明 $e^x + \ln x$、$e^x - \ln x$ 类不等式

【例题】

已知函数 $f(x) = e^x + \ln\dfrac{e}{x}$，证明：$f(x) > 3$。

【证明】

化简：$f(x) = e^x + 1 - \ln x$，即证 $e^x - \ln x > 2$。

方法（切线放缩）：

  • 利用不等式 $e^x \geq x + 1$（当且仅当 $x = 0$ 取等）
  • 利用不等式 $\ln x \leq x - 1$（当且仅当 $x = 1$ 取等），即 $-\ln x \geq 1 - x$

两式相加：$e^x - \ln x \geq (x + 1) + (1 - x) = 2$。

由于取等条件不同（$x=0$ 与 $x=1$ 不能同时成立），故严格大于：$e^x - \ln x > 2$，即 $f(x) > 3$。

09

端点效应失效处理

切入点与着手点

  • 1. 失效情况：$f(a) = 0$ 且 $f'(a) = 0$，端点效应无法给出有效信息
  • 2. 处理方法：对 $f'(x)$ 再求导，研究 $f'(x)$ 的单调性
  • 3. 若 $f'(x)$ 有最小值且最小值 $\geq 0$，则 $f(x)$ 单调递增
  • 4. 本质是"二阶导数"思想的应用

【例题】

已知 $f(x) = (a\ln x - 1)(x - 1) + \ln x$ （$a > 0$），当 $a = \dfrac{1}{2}$ 时，求 $f(x)$ 的单调性。

【答案】在 $(0, +\infty)$ 上单调递增

解析：当 $a = \dfrac{1}{2}$ 时：

$f(x) = \dfrac{1}{2}(x+1)\ln x - x + 1$

求导：$f'(x) = \dfrac{1}{2}\ln x + \dfrac{x+1}{2x} - 1 = \dfrac{1}{2}\left(\ln x + \dfrac{1}{x} - 1\right)$

令 $g(x) = \ln x + \dfrac{1}{x} - 1$，则 $g'(x) = \dfrac{1}{x} - \dfrac{1}{x^2} = \dfrac{x-1}{x^2}$。

当 $0 < x < 1$ 时，$g'(x) < 0$；当 $x > 1$ 时，$g'(x) > 0$。

故 $g(x)$ 在 $x = 1$ 处取最小值 $g(1) = 0$，即 $g(x) \geq 0$ 恒成立。

所以 $f'(x) = \dfrac{1}{2}g(x) \geq 0$，函数在定义域上单调递增（注意：$f'(1)=0$ 但不改变单调性）。

10

隐零点问题

切入点与着手点

  • 1. 隐零点：$f'(x_0) = 0$ 但 $x_0$ 无法显式求出
  • 2. 处理方法：从 $f'(x_0) = 0$ 中解出某个表达式，代入 $f(x_0)$
  • 3. 利用零点关系式消去复杂项，简化 $f(x_0)$ 的表达式
  • 4. 最后利用 $x_0$ 的范围估计 $f(x_0)$ 的正负

【例题】

已知函数 $f(x) = x + \ln x$。证明：$xf(x) + e^{-x} - x \geq 0$。

【证明】

不等式即 $x^2 + x\ln x - x + e^{-x} \geq 0$（$x > 0$）。

设 $g(x) = x^2 + x\ln x - x + e^{-x}$，$g'(x) = 2x + \ln x - e^{-x}$。

观察：$g'(1) = 2 - \dfrac{1}{e} > 0$，$g'\left(\dfrac{1}{2}\right) = 1 - \ln 2 - \dfrac{1}{\sqrt{e}} < 0$。

故存在 $x_0 \in \left(\dfrac{1}{2}, 1\right)$ 使得 $g'(x_0) = 0$，即 $e^{-x_0} = 2x_0 + \ln x_0$（此为隐零点关系式）。

$g(x)$ 在 $x_0$ 处取最小值：

$g(x_0) = x_0^2 + x_0\ln x_0 - x_0 + 2x_0 + \ln x_0 = x_0^2 + x_0 + (x_0+1)\ln x_0$

因 $x_0 \in \left(\dfrac{1}{2}, 1\right)$，利用 $\ln x_0 > -1$ 放缩可知 $g(x_0) > 0$，故原不等式成立。

11

极值点偏移

切入点与着手点

  • 1. 问题特征：$f(x_1) = f(x_2)$，$x_1 \neq x_2$，证明 $x_1 + x_2$ 与 $2x_0$ 的大小关系（$x_0$ 为极值点）
  • 2. 核心思想：设 $x_1 < x_0 < x_2$，利用单调性将 $x_2$ 与 $2x_0 - x_1$ 比较
  • 3. 构造函数：$g(x) = f(x) - f(2x_0 - x)$，证明其单调性
  • 4. 利用 $g(x_0) = 0$ 和单调性得出结论

【例题】

函数 $f(x) = xe^{-x}$ （$x \in \mathbb{R}$），若 $f(x_1) = f(x_2)$ 且 $x_1 \neq x_2$，求证：$x_1 + x_2 > 2$。

【证明】

$f(x)$ 在 $(-\infty, 1)$ 递增，在 $(1, +\infty)$ 递减，极大值点 $x = 1$。

设 $x_1 < 1 < x_2$，要证 $x_2 > 2 - x_1$（注意 $2-x_1 > 1$）。

因 $f(x)$ 在 $(1, +\infty)$ 递减，只需证 $f(x_2) < f(2-x_1)$，即 $f(x_1) < f(2-x_1)$。

构造 $g(x) = f(x) - f(2-x)$（$x < 1$），则 $g(1) = 0$。

$g'(x) = f'(x) + f'(2-x) = (1-x)e^{-x} + (x-1)e^{x-2} = (1-x)(e^{-x} - e^{x-2})$。

当 $x < 1$ 时，$1-x > 0$ 且 $e^{-x} > e^{x-2}$（因 $-x > x-2$），故 $g'(x) > 0$。

所以 $g(x)$ 在 $(-\infty, 1)$ 递增，$g(x) < g(1) = 0$，即 $f(x) < f(2-x)$，得证。

12

导数同构

切入点与着手点

  • 1. 同构思想：将不等式两边化为同一函数 $g(u)$ 的形式
  • 2. 常见同构函数：$g(t) = te^t$、$g(t) = t + e^t$、$g(t) = t + \ln t$
  • 3. 操作步骤：变形 → 识别同构 → 利用单调性比较自变量
  • 4. 注意同构函数的单调区间，确保比较有效

【例题】

函数 $f(x) = me^{mx} - \ln x$，当 $x > 0$ 时，$f(x) > 0$ 恒成立，则 $m$ 的范围为__________。

【答案】$\left(\dfrac{1}{e}, +\infty\right)$

解析：当 $m \leq 0$ 时，$me^{mx} \leq 0$，而 $-\ln x$ 在 $x > 1$ 时为负，不满足，故 $m > 0$。

不等式变形：$me^{mx} > \ln x$。

两边同乘 $x$（$x>0$）：$mxe^{mx} > x\ln x = (\ln x)e^{\ln x}$。

构造函数 $g(t) = te^t$，则不等式为 $g(mx) > g(\ln x)$。

因 $g'(t) = (t+1)e^t$，当 $t > -1$ 时 $g(t)$ 单调递增。

当 $x \geq 1$ 时，$\ln x \geq 0$，只需 $mx > \ln x$，即 $m > \dfrac{\ln x}{x}$。易知 $\dfrac{\ln x}{x}$ 最大值为 $\dfrac{1}{e}$（在 $x=e$ 处），故 $m > \dfrac{1}{e}$。

当 $0 < x < 1$ 时，$\ln x < 0$，而 $mx > 0$，故 $g(mx) > 0 > g(\ln x)$，恒成立。

综上，$m > \dfrac{1}{e}$。

导数大题解题思路总结

第一问常见类型

  • - 求单调区间：求导 → 解 $f'(x)=0$ → 列表
  • - 求极值/最值：找极值点 → 比较端点和极值
  • - 求切线方程：点斜式，注意"在"与"过"

第二问常见类型

  • - 恒成立求参：分离参数 → 求最值
  • - 证明不等式：构造函数 → 求最值
  • - 零点问题：转化为函数图像与 $x$ 轴交点

高频技巧

  • - 端点效应：$f(a)=0$ 时检验 $f'(a)$
  • - 切线放缩：$e^x \geq x+1$，$\ln x \leq x-1$
  • - 同构变换：化为 $g(u_1) \lessgtr g(u_2)$

易错提醒

  • - 定义域是一切讨论的前提
  • - 分离参数时注意分母正负
  • - 极值点处 $f'(x)=0$ 但不一定变号
