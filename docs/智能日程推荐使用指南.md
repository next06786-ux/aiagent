# 智能日程推荐系统使用指南

## 📋 功能概述

智能日程推荐系统基于用户的历史数据、习惯和生产力曲线，为用户推荐最佳的时间安排。

### 核心功能

1. **单任务时间推荐** - 为单个任务推荐最佳执行时间
2. **每日日程生成** - 自动生成一天的完整日程安排
3. **每周日程规划** - 生成一周的日程计划
4. **时间模式分析** - 分析用户的时间使用习惯和生产力曲线

## 📡 API接口说明

### 1. 推荐单个任务的最佳时间

**接口**: `POST /api/v5/schedule/recommend-time`

**请求示例**:
```json
{
  "user_id": "user_001",
  "task": {
    "task_id": "task_1",
    "title": "完成项目报告",
    "task_type": "work",
    "duration_minutes": 120,
    "priority": "high",
    "deadline": "2026-04-07T18:00:00",
    "flexibility": 0.3,
    "energy_required": 0.8,
    "focus_required": 0.9
  },
  "num_alternatives": 3
}
```

**任务类型** (`task_type`):
- `work` - 工作任务
- `study` - 学习任务
- `exercise` - 运动任务
- `social` - 社交活动
- `rest` - 休息

**优先级** (`priority`):
- `high` - 高优先级
- `medium` - 中优先级
- `low` - 低优先级

## 🎯 推荐算法说明

### 评分机制

系统综合考虑以下因素为时间段评分：

1. **生产力曲线** (40%) - 基于历史数据的生产力水平
2. **能量匹配度** (20%) - 任务所需能量与时段能量的匹配
3. **专注度匹配** (20%) - 任务所需专注度与时段专注度的匹配
4. **时间灵活度** (10%) - 任务的时间灵活性
5. **优先级加权** - 高优先级任务获得额外加分
6. **截止时间压力** - 临近截止时间的任务优先安排

### 任务类型适配

- **工作/学习任务**: 优先安排在高生产力、高专注度时段（如上午9-11点）
- **运动任务**: 优先安排在高能量时段（如早晨7-8点、傍晚5-6点）
- **社交活动**: 优先安排在晚上时段（如晚上6-10点）
- **休息**: 安排在低生产力时段（如午后1-2点）

### 智能优化

- 自动在长时间工作后插入休息时间
- 根据任务优先级和截止时间动态调整
- 考虑用户的个人习惯和偏好
- 提供多个备选方案供用户选择

## 💡 使用建议

### 1. 任务设置技巧

- **准确设置任务类型**: 不同类型的任务会被安排在不同的时段
- **合理设置优先级**: 高优先级任务会优先获得最佳时段
- **设置截止时间**: 有助于系统更好地规划任务顺序
- **调整灵活度**: 灵活度高的任务可以被安排在次优时段

### 2. 提高推荐准确性

- **持续使用**: 系统会学习你的习惯，推荐会越来越准确
- **反馈调整**: 根据实际执行情况调整任务参数
- **定期查看模式分析**: 了解自己的时间使用习惯

### 3. 日程优化建议

- **遵循生产力曲线**: 在高效时段做重要的事
- **劳逸结合**: 注意工作和休息的平衡
- **保持规律**: 建立稳定的作息习惯
- **留有余地**: 不要把日程排得太满
