## 📋 技术债务和优化机会

### 高优先级
- [ ] LoRA 增量量化
  - 新训练的 LoRA 直接量化（无原始版本保存）
  - 节省存储空间 50%

- [ ] 自适应量化
  - 根据推理负载自动调整量化策略
  - 推理快速路径检测

- [ ] 多 GPU 分布式量化
  - 并行量化多个 LoRA
  - 加速 4 倍

### 中优先级
- [ ] 量化感知训练（QAT）
  - 从头训练时引入量化模拟
  - 精度提升 2-3%

- [ ] 混合精度量化
  - 关键层 8-bit，其他层 4-bit
  - 精度 +1%，推理速度 +5%

- [ ] 自动后端选择
  - 基于硬件和推理负载
  - 动态切换最优方案

### 低优先级
- [ ] 支持三比特量化（2-bit）
  - llmquant 已支持
  - 进一步压缩但精度损失

- [ ] 权重聚类量化
  - K-means 聚类 → 量化码本
  - 适合离线场景
