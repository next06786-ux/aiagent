- 不会被封禁
- 数据质量高

#### 申请流程
1. 访问 BOSS直聘开放平台
2. 注册开发者账号
3. 申请API权限
4. 获取 API Key


## 数据流向

```
真实招聘网站
  ↓
real_job_data_integration.py (数据获取+缓存)
  ↓
multi_agent_career_evaluator.py (Agent使用数据)
  ├─ SkillDevelopmentAgent: 从岗位JD提取技能要求
  ├─ MarketEnvironmentAgent: 统计岗位数量、薪资趋势
  └─ FinancialStatusAgent: 使用真实薪资数据计算ROI
  ↓
career_simulation_integration.py (生成推演)
  ↓
前端展示
```

### 3. 在Agent中使用
```python
# 在 SkillDevelopmentAgent.initialize() 中
from backend.vertical.career.real_job_data_integration import real_job_integration

jobs = real_job_integration.search_jobs(
    keyword=target_role,
    location="北京",
    limit=20
)

# 从真实岗位提取技能要求
skill_freq = {}
for job in jobs:
    for skill in job.required_skills:
        skill_freq[skill] = skill_freq.get(skill, 0) + 1

# 取高频技能作为目标
self.target_skills = {
    skill: 8.0
    for skill, freq in skill_freq.items()
    if freq >= len(jobs) * 0.3  # 出现在30%以上的岗位
}
```

## 定时更新任务

### 使用APScheduler
```python
from apscheduler.schedulers.background import BackgroundScheduler

def update_job_data():
    """定时更新岗位数据"""
    from backend.vertical.career.real_job_data_integration import real_job_integration
    
    keywords = ["Python工程师", "Java工程师", "产品经理", "数据分析师"]
    cities = ["北京", "上海", "深圳", "杭州"]
    
    for keyword in keywords:
        for city in cities:
            try:
                jobs = real_job_integration.search_jobs(keyword, city, 50)
                # 存入Neo4j
                save_jobs_to_neo4j(jobs)
                print(f"更新 {keyword} @ {city}: {len(jobs)} 个岗位")
            except Exception as e:
                print(f"更新失败: {e}")

# 启动定时任务
scheduler = BackgroundScheduler()
scheduler.add_job(update_job_data, 'cron', hour=2)  # 每天凌晨2点更新
scheduler.start()
```

## 常见问题

### Q: 爬虫被封了怎么办？
A: 
1. 降低请求频率
2. 使用代理IP
3. 切换到官方API
4. 使用缓存数据

### Q: 数据不准确怎么办？
A:
1. 多数据源交叉验证
2. 人工抽查数据质量
3. 设置数据异常检测

### Q: 如何保证数据实时性？
A:
1. 设置合理的缓存时间（24小时）
2. 定时任务更新热门岗位
3. 用户查询时触发更新
