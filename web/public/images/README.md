# 图片资源目录

## 主页展示图片

### 1. 视频演示区域图片

将你的展示图片命名为 `showcase.png` 并放在这个目录下。

要求：
- 文件名：`showcase.png`（或 `showcase.jpg`）
- 推荐尺寸：1920x1080 或 16:9 比例
- 格式：PNG、JPG 或 WEBP
- 大小：建议不超过 2MB

### 2. 智慧洞察界面图片（电脑展示区域）

将智慧洞察的3Agent界面截图命名为 `insights-dashboard.png` 并放在这个目录下。

要求：
- 文件名：`insights-dashboard.png`（或 `insights-dashboard.jpg`）
- 推荐尺寸：1920x1080 或 16:9 比例
- 格式：PNG、JPG 或 WEBP
- 大小：建议不超过 2MB
- 内容：智慧洞察的3Agent界面（教育升学、职业发展、人际关系）

### 使用方法：
1. 将图片复制到 `web/public/images/` 目录
2. 重命名为对应的文件名
3. 刷新页面即可看到效果

如果图片不存在，页面会显示默认的占位符或仪表盘界面。
